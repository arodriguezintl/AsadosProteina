# Manual de Usuario - Base
**Sistema ERP Asados Proteína**

## Introducción
Bienvenido al sistema de gestión integral para Asados Proteína. Este manual describe los módulos principales y cómo utilizarlos en el día a día.

## 1. Acceso y Roles
El sistema utiliza roles para asegurar que cada usuario vea solo lo que necesita.
- **Cajero**: Acceso al POS y Ventas.
- **Cocinero**: Acceso a Recetas y Órdenes de Cocina.
- **Repartidor**: Acceso a modulo de Entregas.
- **Admin/Super Admin**: Acceso total a finanzas, RRHH y configuración.

## 2. Módulos del Sistema

### 2.1 Punto de Venta (POS)
El módulo principal para la operación diaria.
- **Crear Orden**: Seleccione productos del menú.
- **Pago**: Procese pagos en efectivo o tarjeta.
- **Nota**: Al confirmar una orden, el inventario se descuenta automáticamente.

### 2.2 Inventario
Gestión de existencias.
- **Productos**: Alta, baja y modificación de productos (terminados o materia prima).
- **Categorías**: Organización de productos.
- **Movimientos**: Historial de entradas y salidas de stock.

### 2.3 Recetas
Gestión de costos y preparación.
- **Ingredientes**: Asocie productos de inventario a un platillo.
- **Costo**: El sistema calcula el costo por porción basado en los ingredientes.

### 2.4 Recursos Humanos
- **Empleados**: Registro de personal.
- **Asistencia**: Control de entradas y salidas.

## 3. Soporte
Para problemas técnicos, contacte al administrador del sistema o al equipo de desarrollo.
